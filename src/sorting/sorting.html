Algorithm Info

Bubble Sort

Time: O(n²) | Space: O(1)

Repeatedly swaps adjacent elements if they are in the wrong order

Selection Sort

Time: O(n²) | Space: O(1)

Finds minimum element and places it at the beginning

Insertion Sort

Time: O(n²) | Space: O(1)

Builds sorted array one element at a time

Merge Sort

Time: O(n log n) | Space: O(n)

Divides array, sorts, and merges back together

Quick Sort

Time: O(n log n) | Space: O(log n)

Picks a pivot and partitions array around it
